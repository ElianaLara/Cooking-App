# Cooking Saver

**Cooking Saver** is a web-based app designed to help users **create, manage, and organize recipes** easily. The app features an engaging start screen and a CRUD (Create, Read, Update, Delete) system for managing recipes such as vegetarian, sweet, or vegan dishes.

---

## Features

-  **Interactive Start Screen** – Welcomes users before accessing their recipe collection.
-  **Recipe Management (CRUD)** – Add, edit, and delete your favourite recipes.
-  **Search & Filter** – Filter recipes by category (Vegetarian, Sweet, Vegan).
-  **Dynamic Form** – Input recipe title, ingredients, and instructions.
-  **Persistent UI** – Clean and responsive interface with Font Awesome icons.

---

##  Technologies Used

- **HTML5** – Structure and layout
- **CSS3** – Custom styles (`start.css`) and pixel-art aesthetic
- **JavaScript (app.js)** – Logic for CRUD operations and UI toggling
- **Font Awesome** – For button and UI icons
- **Custom Font** – “Minecraft” style for a retro feel

## ⚙️ How to Run

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Click **Start** to access your recipe manager.
4. Add, edit, or delete recipes from your personal collection!
